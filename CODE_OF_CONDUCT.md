# Code of Conduct

This project follows the Contributor Covenant v2.1 â€” a standard, community-friendly code of conduct that helps keep our community welcoming and respectful.

The short version:

- Be respectful and kind.
- Be considerate of others' time and effort.
- If you see harassment or unacceptable behavior, report it to the maintainers.

Expected behavior

- Use welcoming and inclusive language.
- Be respectful in discussions and code review comments.
- Give constructive feedback and assume good faith.

Unacceptable behavior

- Harassment, hate speech, or discriminatory language.
- Personal attacks, doxxing, or threats.
- Deliberate disruption of the project or community.

Reporting

If you witness or experience unacceptable behavior, please report it by opening an issue in this repository with the title "Code of Conduct Report" and a clear description of the incident.

Enforcement

Reports will be reviewed by the project maintainers and handled appropriately. Enforcement may include warnings, temporary or permanent bans from contributions, or notifying platform administrators.
